# 🛁 Fibras & Jacuzzis – Sitio Web Corporativo

Sitio web corporativo para **Fibras & Jacuzzis**, empresa dedicada a la fabricación y comercialización de **jacuzzis, piscinas, lavaderos y rodaderos en fibra de vidrio**.

El proyecto está desarrollado con **HTML, CSS y JavaScript puro**, enfocado en diseño moderno, rendimiento y facilidad de mantenimiento.

---

## 🚀 Características

- Diseño moderno y totalmente responsive
- Visor de imágenes interactivo (clic para cambiar imagen principal)
- Catálogo de productos con filtros por categoría
- Integración directa con WhatsApp para cotizaciones
- Sección de ubicación con Google Maps embebido
- Menú hamburguesa para dispositivos móviles
- Código limpio, comentado y fácil de personalizar
- Sin frameworks pesados (no Bootstrap, no Swiper)

---

## 🧩 Tecnologías utilizadas

- HTML5  
- CSS3 (Flexbox, Grid, Media Queries)  
- JavaScript Vanilla  
- Google Maps Embed  
- Google Fonts  
- Font Awesome
